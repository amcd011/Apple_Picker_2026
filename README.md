# Apple Picker — Starter Code

This repository contains **starter code** for the Apple Picker game (COSC-3550).  
You are **not expected to push changes back to this repository**.

Choose **one** of the following workflows depending on how you want to work.

---

## Option 1 — Download as ZIP (Quickest)

**Recommended if you don't need Git version control and just want the files.**

1. Go to <https://github.com/COSC-3550/Apple_Picker_2026>
2. Click the green **Code** button.
3. Select **Download ZIP**.
4. Extract the ZIP to a folder on your computer.
5. Open the project folder in Unity Hub.

> You can submit your finished project by re-zipping the folder and uploading it to D2L.

---

## Option 2 — Clone and Work Locally (No Git Push Required)

**Recommended if you want to use Git locally but don't need to push to GitHub.**

You will:

- Clone the repository to your computer
- Edit and run the project locally
- ❌ **Not** push changes back to GitHub (you do not have write access)

### Option 2A — Command Line

```bash
git clone https://github.com/COSC-3550/Apple_Picker_2026.git
cd Apple_Picker_2026
```

Open the project folder in Unity Hub and work normally.

### Option 2B — GitHub Desktop

1. Open **GitHub Desktop**.
2. Click **File → Clone Repository**.
3. Select the **URL** tab.
4. Paste: `https://github.com/COSC-3550/Apple_Picker_2026`
5. Choose a local folder and click **Clone**.

---

## Option 3 — Fork then Clone (Full Git Workflow)

**Recommended if you want full Git version control and the ability to push commits to your own repository.**

You will:

- Create your own copy of the repository on GitHub (a **fork**)
- Clone your fork to your computer
- Push changes to **your own** repository only

### Step 1 — Fork the Repository

1. Go to <https://github.com/COSC-3550/Apple_Picker_2026>
2. Click **Fork** (top-right corner).
3. Select your own GitHub account as the destination.

You now have a copy at `https://github.com/YOUR_GITHUB_USERNAME/Apple_Picker_2026`.

### Step 2 — Clone Your Fork

#### Command Line

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/Apple_Picker_2026.git
cd Apple_Picker_2026
```

#### GitHub Desktop

1. Open **GitHub Desktop**.
2. Click **File → Clone Repository**.
3. Select the **URL** tab.
4. Paste: `https://github.com/YOUR_GITHUB_USERNAME/Apple_Picker_2026`
5. Choose a local folder and click **Clone**.

### Step 3 — Push Changes to Your Fork

Since you own the fork, you can commit and push freely:

```bash
git add .
git commit -m "Your commit message"
git push origin main
```

> ⚠️ Do **not** open a Pull Request back to the original repository unless specifically instructed.

---

## Opening the Project in Unity

1. Open **Unity Hub**.
2. Click **Open** (or **Add**) and browse to the project folder you cloned or extracted.
3. Make sure you are using **Unity 2022** (the version specified for this course).
4. Click **Open** to load the project.
