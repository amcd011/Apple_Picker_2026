# Apple Picker — Starter Code

This repository contains **starter code** for the Apple Picker game.  
You are **not expected to push changes back to this repository**.

Choose **one** of the following workflows depending on how you want to work.

---

## Option 1 — Clone and Work Locally (No Git Push Required)
**Recommended if you just want to code locally and submit your work elsewhere (ZIP, LMS, etc.).**

You will:
- Clone the repository to your computer
- Edit and run the project locally
- ❌ NOT push changes back to GitHub

### Option 1A — Command Line

```bash
git clone https://github.com/COSC-3550/Apple_Picker_2026
cd Apple_Picker_2026
bash```

You can now open the project in Unity and work normally.
- Do not run git push. You do not have write access to this repository.

### Option 1B - GitHub Desktop
1. Open GitHub Desktop
2. Click File -> Clone Repository
3. Select URL
4. Paste: https://github.com/COSC-3550/Apple_Picker_2026
5. Choose a local folder and click Clone.


## Option 2 - Fork then Clone (Full Git Workflow)
**Recommened if you want full git control

You will:
- Create your own copy of the repository on GitHub (a fork)
- Clone your fork to your computer
- Push changes to your own repository only

### Step 1 - Fork the Repository
1. Go to https://github.com/COSC-3550/Apple_Picker_2026 
2. Click Fork (top right)
3. Select your own GitHub account

### Step 2A - Clone your fork (Command Line)
```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/Apple_Picker_2026.git
cd Apple_Picker_2026
